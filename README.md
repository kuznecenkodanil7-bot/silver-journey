# Minesweeper Client для Minecraft 1.21.5

Клиентский Fabric-мод: по нажатию клавиши **Ж** открывается экран мини-игры «Сапёр».

## Что есть

- Открытие по клавише **Ж**. Технически это `GLFW_KEY_SEMICOLON`, то есть физическая клавиша `;` на US-раскладке / `Ж` на русской раскладке.
- Три сложности:
  - Лёгкий: 9×9, 10 мин
  - Средний: 16×16, 40 мин
  - Сложный: 30×16, 99 мин
- ЛКМ открывает клетку, ПКМ ставит/снимает флаг.
- Первый клик безопасный: мины расставляются после первого клика и не попадают в область 3×3 вокруг него.
- Экран не ставит игру на паузу.

## Сборка

Нужно: JDK 21 и Gradle/IntelliJ IDEA с Gradle.

```bash
gradle build
```

Готовый `.jar` появится в:

```text
build/libs/minesweeper-client-1.0.0.jar
```

Положи этот `.jar` в папку `mods` клиента Minecraft 1.21.5 с Fabric Loader и Fabric API.

## Версии в проекте

- Minecraft: `1.21.5`
- Yarn mappings: `1.21.5+build.1`
- Fabric Loader: `0.16.10`
- Fabric API: `0.119.5+1.21.5`
- Fabric Loom: `1.10-SNAPSHOT`
